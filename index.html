The Box Model in Action

This box visually demonstrates the core components of the CSS Box Model: content, padding, border, and margin. The light blue area is the padding, the dark blue line is the border, and the space around the entire box is the margin.

This section is styled like a card. It uses padding to create space between the text and the border. The margin below it ensures there's a clear separation from the next element.
